Primeiro Projeto na Alura

Neste projeto da Alura - utilizei várias ferramentas do Html e CSS, fiz 3 páginas que navegam entre si. A primeira página foi desenvolvida com Informações sobre quem era o Cliente no caso a Barbearia Alura, mostrando ao usuário de forma visual uma imagem. Outra imagem representa um mapa informando a localização da Barbearia. Esteve presente as informações de "missão" e "benefícios". Um breve vídeo foi inserido para demostração do serviço. Finalizando com uma marca registrada da empresa em um rodapé com imagem.

Tecnologias utilizadas neste Projeto HTML:

  • Body, Header, div, h1, img, nav, ul, li, a href, main, section, h2, p, strong, h3, iframe, footer.

Estilização no CSS:

  • Margin, width, background, padding-left, text-align: center, font-size, color, display: inline-block, vertical-align:top, margin-right.
